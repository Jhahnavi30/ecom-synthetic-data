# Synthetic Ecommerce Data Generator

This repository contains a utilities suite for generating synthetic datasets that mimic ecommerce activity for demos, analytics sandboxes, and testing pipelines.

## Features
- Parameterized dataset generation for customers, orders, products, and browsing sessions
- Pluggable data schemas and scenario presets
- CSV and JSON output with deterministic seeding for reproducibility

## Getting Started
1. Clone the repository: `git clone https://github.com/<your-username>/ecom-synthetic-data.git`
2. Install dependencies (add your preferred tooling stack here).
3. Run the generator scripts or notebooks to produce sample datasets.
